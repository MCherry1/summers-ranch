# Summers Ranch Website

Family-owned cattle breeding operation. Hosted on GitHub Pages at [mrsummersranch.com](https://mrsummersranch.com).

## Quick Start

Static site — pure HTML, CSS, and vanilla JavaScript. No build tools needed.

- Push to `main` → GitHub Pages auto-deploys in ~2 minutes
- Custom domain via CNAME + Porkbun DNS
- `mrsranch.com` redirects here via Porkbun URL forwarding

## Structure

```
index.html          Home page
about.html          Our Story
cattle.html         Our Cattle (filterable grid)
gallery.html        Photo & Video Gallery
contact.html        Contact page with form & map
images/             All site photos (organized by subfolder)
site-config.json    Content configuration reference
```
