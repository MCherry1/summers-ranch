## Nudge system

### High priority
- [ ] Sale animal with stale photos (>60 days)
- [ ] Sale animal with no photos
- [ ] Pregnancy overdue (>300 days from `pregnancySetDate` or >14 days past `expectedCalving`)

### Medium priority
- [ ] Summary: "N animals need basic details" (born, sex, breed)
- [ ] Only counts animals past 7-day grace period
- [ ] Multiple calves from same dam (needs retagging)

### Low priority
- [ ] Stale photos (>180 days, non-sale)
- [ ] Missing side profile classification
- [ ] Missing head shot classification
- [ ] Missing three-quarter classification

### Persistent (can't be permanently dismissed)
- [ ] Unbranded active herd animals
- [ ] Returns after 24-hour dismissal
- [ ] Doesn't count calves still nursing (branded at weaning)

### Dismissal
- [ ] localStorage-based with 24-hour timestamp
- [ ] Entire nudge block hidden when dismissed
- [ ] Returns in full after 24 hours
